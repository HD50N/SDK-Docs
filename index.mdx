A powerful JavaScript SDK for recording user sessions and tracking behavior analytics in web applications. Built on top of [rrweb](https://github.com/rrweb-io/rrweb) for high-fidelity session recordings.

## Quick Start

<Info>
**Framework Guides:** If you're using Next.js, React, or another framework, we recommend checking out our [framework guides](/essentials/frameworks) for optimized setup instructions.
</Info>

### 1. Install the SDK

```bash
npm install humanbehavior-js
```

<Info>
<strong>How to find your Public API Key:</strong><br/>
<b>Flow:</b> Project Home → Settings (on sidebar) → Public API Key
</Info>

<p style={{ fontSize: '0.95em', color: '#888', marginTop: '-0.5rem', marginBottom: '1.5rem' }}><em>Example screenshot – your actual API key will be different.</em></p>

### 2. Initialize the Tracker

```javascript
import { HumanBehaviorTracker } from 'humanbehavior-js';

// One-line initialization with global persistence
const tracker = HumanBehaviorTracker.init('your-api-key', {
  redactFields: ['input[type="password"]']
});
```

<Note>
**That's it!** The SDK automatically tracks user interactions, navigation, and console events. You don't need to add any custom event tracking unless you want to track specific business events.
</Note>

### 3. Add User Information (Optional)

```javascript
// Add user information when available
await tracker.addUserInfo({
  email: 'user@example.com',
  name: 'John Doe'
});
```

## Key Features

The SDK automatically provides:

- **Session Recording**: Mouse movements, clicks, scrolls, form inputs
- **Automatic Event Tracking**: Button clicks, link clicks, form submissions
- **Navigation Tracking**: SPA route changes, browser navigation
- **Console Tracking**: Log, warn, error messages
- **Custom Events**: Track specific user actions (optional)
- **Data Redaction**: Protect sensitive information
- **Session Continuity**: Persists across page reloads

## What Gets Recorded

The SDK automatically captures:

- **Mouse movements** and clicks
- **Keyboard input** and form interactions  
- **Scroll events** and viewport changes

## Session Persistence

Sessions automatically persist for 30 minutes of inactivity:

```javascript
// Session continues across:
// - Page refreshes within 30 minutes
// - Tab switching and reopening
// - Short periods of inactivity

// New session created after:
// - 30+ minutes of inactivity
// - Manual session reset
// - Browser restart
```

## React Integration

For React applications, use the dedicated React integration:

```javascript
import { HumanBehaviorProvider, useHumanBehavior } from 'humanbehavior-js/react';

function App() {
  return (
    <HumanBehaviorProvider apiKey="your-api-key">
      <YourApp />
    </HumanBehaviorProvider>
  );
}

function MyComponent() {
  const tracker = useHumanBehavior();
  
  const handleClick = () => {
    // Your business logic here
    console.log('Button clicked!');
    // Custom events are optional - the SDK automatically tracks clicks
  };
  
  return <button onClick={handleClick}>Submit</button>;
}
```

## Data Privacy & Security

### Automatic Redaction

```javascript
// Redact sensitive fields during recording
tracker.setRedactedFields([
  'input[type="password"]',
  '#email-field',
  '.sensitive-data'
]);
```

### Consent Management

```javascript
// Only track with user consent
if (userConsent) {
  const tracker = HumanBehaviorTracker.init('your-api-key');
} else {
  console.log('Session recording disabled - no consent');
}
```

## Examples

### Vanilla JavaScript

```html
<!DOCTYPE html>
<html>
<head>
  <script src="https://unpkg.com/humanbehavior-js@latest/dist/index.min.js"></script>
</head>
<body>
  <button id="submit-btn">Submit</button>
  
  <script>
    // Initialize tracker with automatic tracking
    const tracker = HumanBehaviorTracker.init('your-api-key');
    
    // Button clicks are automatically tracked!
    // No manual event listeners needed
    
    // Your business logic here
    document.getElementById('submit-btn').addEventListener('click', () => {
      console.log('Processing submission...');
      // The SDK automatically tracks button clicks
      // Custom events are optional for additional business tracking
    });
  </script>
</body>
</html>
```

### React Application

```javascript
import { HumanBehaviorProvider, useHumanBehavior } from 'humanbehavior-js/react';

function App() {
  return (
    <HumanBehaviorProvider apiKey="your-api-key">
      <YourApp />
    </HumanBehaviorProvider>
  );
}

function ContactForm() {
  const tracker = useHumanBehavior();
  
  const handleSubmit = (formData) => {
    // Your form processing logic here
    console.log('Form submitted!');
    // The SDK automatically tracks form submissions
    // Custom events are optional for additional business tracking
  };
  
  return (
    <form onSubmit={handleSubmit}>
      <input name="name" placeholder="Name" />
      <input name="email" placeholder="Email" />
      <textarea name="message" placeholder="Message" />
      <button type="submit">Submit</button>
    </form>
  );
}
```
