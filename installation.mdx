# Installation

HumanBehavior SDK can be installed in several ways depending on your project setup and preferences.

## Package Managers

### NPM

```bash
npm install humanbehavior-js
```

### Yarn

```bash
yarn add humanbehavior-js
```

### PNPM

```bash
pnpm add humanbehavior-js
```

## CDN

For quick prototyping or when you can't use a package manager:

```html
<script src="https://unpkg.com/humanbehavior-js@latest/dist/index.min.js"></script>
```

<Note>
  The CDN version exposes `HumanBehaviorTracker` globally on the `window` object.
</Note>

## Local File

If you have the SDK files locally (e.g., after building from source):

```html
<script src="./dist/index.min.js"></script>
```

<Note>
  Make sure the path to your `dist/index.min.js` file is correct relative to your HTML file.
</Note>

## Module Types

The SDK supports multiple module formats:

### ES Modules (ESM)

```javascript
import { HumanBehaviorTracker } from 'humanbehavior-js';
```

### CommonJS (CJS)

```javascript
const { HumanBehaviorTracker } = require('humanbehavior-js');
```

### TypeScript

The SDK includes TypeScript definitions:

```typescript
import { HumanBehaviorTracker } from 'humanbehavior-js';

const tracker: HumanBehaviorTracker = new HumanBehaviorTracker('your-api-key');
```

## Framework Integration

### React

For React applications, import the React integration:

```javascript
import { HumanBehaviorProvider, useHumanBehavior } from 'humanbehavior-js/react';
```

### Next.js

The SDK is fully compatible with Next.js and server-side rendering:

```javascript
// pages/_app.js
import { HumanBehaviorProvider } from 'humanbehavior-js/react';

export default function App({ Component, pageProps }) {
  return (
    <HumanBehaviorProvider apiKey={process.env.NEXT_PUBLIC_HUMANBEHAVIOR_API_KEY}>
      <Component {...pageProps} />
    </HumanBehaviorProvider>
  );
}
```

### Vue.js

Use the standard JavaScript SDK with Vue:

```javascript
// main.js
import { createApp } from 'vue';
import { HumanBehaviorTracker } from 'humanbehavior-js';
import App from './App.vue';

const app = createApp(App);

// Initialize tracker
const tracker = new HumanBehaviorTracker('your-api-key');

// Make tracker available globally
app.config.globalProperties.$tracker = tracker;

app.mount('#app');
```

## Environment Variables

For security, store your API key in environment variables:

### Development (.env)

```bash
HUMANBEHAVIOR_API_KEY=your-api-key-here
NEXT_PUBLIC_HUMANBEHAVIOR_API_KEY=your-api-key-here
```

### Production

```bash
# Set environment variables in your hosting platform
HUMANBEHAVIOR_API_KEY=your-production-api-key
NEXT_PUBLIC_HUMANBEHAVIOR_API_KEY=your-production-api-key
```

## Ingestion Server Configuration

The SDK supports multiple ingestion server configurations:

### Default Configuration

By default, the SDK uses a development server:

```javascript
// Uses default development server
const tracker = new HumanBehaviorTracker('your-api-key');
```

### Custom Ingestion Server

You can specify a custom ingestion server:

```javascript
// Use custom ingestion server
const tracker = new HumanBehaviorTracker(
  'your-api-key',
  'https://your-custom-server.com'
);
```

### Environment-based Configuration

For different environments:

```javascript
const ingestionUrl = process.env.NODE_ENV === 'production' 
  ? 'https://production-ingestion-server.com'
  : 'http://localhost:3000';

const tracker = new HumanBehaviorTracker('your-api-key', ingestionUrl);
```

## Browser Compatibility

The SDK supports all modern browsers:

| Browser | Version |
|---------|---------|
| Chrome | 60+ |
| Firefox | 55+ |
| Safari | 12+ |
| Edge | 79+ |

<Warning>
  Internet Explorer is not supported. The SDK uses modern JavaScript features that are not available in IE.
</Warning>

## Bundle Size

The SDK is optimized for size:

- **Core SDK**: ~50KB (gzipped)
- **With React integration**: ~55KB (gzipped)
- **CDN version**: ~65KB (gzipped)

<Note>
  Version 0.0.8 includes enhanced URL tracking and improved session management, with minimal impact on bundle size.
</Note>

## Dependencies

The SDK has minimal dependencies:

- `rrweb`: Session recording engine
- `uuid`: Unique identifier generation
- `react`: Only for React integration

## Troubleshooting

### Module Not Found

If you encounter module resolution issues:

```bash
# Clear cache and reinstall
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

### Version Compatibility

Make sure you're using a compatible version:

```bash
# Check current version
npm list humanbehavior-js

# Update to latest version
npm update humanbehavior-js
```
