# Quickstart Guide

Get started with HumanBehavior SDK and begin tracking user sessions in just a few minutes.

## Prerequisites

Before you begin, make sure you have:

- A HumanBehavior account and API key
- Your ingestion server URL
- A modern web browser
- Basic knowledge of JavaScript

## Step 1: Install the SDK

Choose your preferred installation method:

<CodeGroup>

```bash npm
npm install humanbehavior-js
```

```bash yarn
yarn add humanbehavior-js
```

```bash pnpm
pnpm add humanbehavior-js
```

</CodeGroup>

## Step 2: Initialize the Tracker

Create a new instance of the tracker with your API key and ingestion URL:

```javascript
import { HumanBehaviorTracker } from 'humanbehavior-js';

const tracker = new HumanBehaviorTracker(
  'your-api-key-here',
  'https://ingestion.humanbehavior.ai'
);
```

    <Note>
  Replace `'your-api-key-here'` with your actual API key from your HumanBehavior dashboard.
    </Note>

## Step 3: Start Recording

Begin recording user sessions:

```javascript
// Start the session recording
tracker.start();
```

## Step 4: Add User Information (Optional)

Identify users for better analytics:

```javascript
// Add user information
await tracker.addUserInfo({
  email: 'user@example.com',
  name: 'John Doe',
  userId: '12345'
});
```

## Complete Example

Here's a complete working example:

```javascript
import { HumanBehaviorTracker } from 'humanbehavior-js';

// Initialize tracker
const tracker = new HumanBehaviorTracker(
  'your-api-key-here',
  'https://ingestion.humanbehavior.ai'
);

// Start recording immediately
tracker.start();

// Add user information when available
async function identifyUser(userData) {
  try {
    await tracker.addUserInfo(userData);
    console.log('User identified successfully');
  } catch (error) {
    console.error('Failed to identify user:', error);
  }
}

// Example usage
identifyUser({
  email: 'user@example.com',
  name: 'John Doe'
});
```

## HTML Example

If you're using the CDN version:

```html
<!DOCTYPE html>
<html>
<head>
    <title>My App</title>
    <script src="https://unpkg.com/humanbehavior-js@latest/dist/index.min.js"></script>
</head>
<body>
    <h1>Welcome to My App</h1>
    
    <script>
        // Initialize tracker
        const tracker = new HumanBehaviorTracker(
            'your-api-key-here',
            'https://ingestion.humanbehavior.ai'
        );
        
        // Start recording
        tracker.start();
        
        // Add user info when form is submitted
        document.getElementById('loginForm').addEventListener('submit', async (e) => {
            e.preventDefault();
            const formData = new FormData(e.target);
            
            await tracker.addUserInfo({
                email: formData.get('email'),
                name: formData.get('name')
            });
        });
    </script>
</body>
</html>
```

Or if you have the SDK files locally:

```html

</html>
```

## What Happens Next?

Once you've completed the setup:

1. **Session Recording**: The SDK automatically captures user interactions
2. **Data Processing**: Events are processed and sent to your ingestion server
3. **User Identification**: Users are automatically identified using cookies
4. **Analytics**: View session recordings and analytics in your HumanBehavior dashboard
