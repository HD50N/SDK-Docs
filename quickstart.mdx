# Quickstart Guide

Get started with HumanBehavior SDK and begin tracking user sessions in just a few minutes.

## Prerequisites

Before you begin, make sure you have:

- A HumanBehavior account and API key
- A modern web browser
- Basic knowledge of JavaScript

## Step 1: Install the SDK

Choose your preferred installation method:

<CodeGroup>

```bash npm
npm install humanbehavior-js
```

```bash yarn
yarn add humanbehavior-js
```

```bash pnpm
pnpm add humanbehavior-js
```

</CodeGroup>

## Step 2: Initialize the Tracker

Use the recommended static `init()` method for one-line initialization:

```javascript
import { HumanBehaviorTracker } from 'humanbehavior-js';

const tracker = HumanBehaviorTracker.init('your-api-key-here');
```

<Note>
  Replace `'your-api-key-here'` with your actual API key from your HumanBehavior dashboard. The SDK will automatically use the appropriate ingestion server based on your configuration.
</Note>

## Step 3: Add User Information (Optional)

Identify users for better analytics:

```javascript
// Add user information
await tracker.addUserInfo({
  email: 'user@example.com',
  name: 'John Doe',
  userId: '12345'
});
```

## Complete Example

Here's a complete working example:

```javascript
import { HumanBehaviorTracker } from 'humanbehavior-js';

// Initialize tracker with options
const tracker = HumanBehaviorTracker.init('your-api-key-here', {
  ingestionUrl: 'http://localhost:3000', // optional custom server
  logLevel: 'debug', // optional: 'none' | 'error' | 'warn' | 'info' | 'debug'
  redactFields: ['input[type="password"]', '#email'] // optional sensitive fields
});

// Add user information when available
async function identifyUser(userData) {
  try {
    await tracker.addUserInfo(userData);
    console.log('User identified successfully');
  } catch (error) {
    console.error('Failed to identify user:', error);
  }
}

// Track custom events
async function trackButtonClick(buttonId) {
  await tracker.customEvent('button_clicked', {
    buttonId: buttonId,
    page: window.location.pathname
  });
}

// Example usage
identifyUser({
  email: 'user@example.com',
  name: 'John Doe'
});

// Track a button click
document.getElementById('submit-btn').addEventListener('click', () => {
  trackButtonClick('submit-btn');
});
```

## HTML Example

If you're using the CDN version:

```html
<!DOCTYPE html>
<html>
<head>
    <title>My App</title>
    <!-- Add the HumanBehavior SDK from unpkg -->
    <script src="https://unpkg.com/humanbehavior-js@latest/dist/index.min.js"></script>
</head>
<body>
    <h1>Welcome to My App</h1>
    <button id="submit-btn">Submit</button>
    
    <script>
        // Initialize tracker with global persistence
        const tracker = HumanBehaviorTracker.init('your-api-key-here', {
            logLevel: 'info',
            redactFields: ['input[type="password"]']
        });
        
        // Add user info when form is submitted
        document.getElementById('loginForm').addEventListener('submit', async (e) => {
            e.preventDefault();
            const formData = new FormData(e.target);
            
            await tracker.addUserInfo({
                email: formData.get('email'),
                name: formData.get('name')
            });
        });

        // Track button clicks
        document.getElementById('submit-btn').addEventListener('click', async () => {
            await tracker.customEvent('button_clicked', {
                buttonId: 'submit-btn',
                page: window.location.pathname
            });
        });
    </script>
</body>
</html>
```

<Note>
  The script tag must be added in the `<head>` section before any JavaScript that uses the SDK. This ensures the `HumanBehaviorTracker` class is available globally.
</Note>

## React Example

For React applications:

```javascript
import { HumanBehaviorProvider, useHumanBehavior } from 'humanbehavior-js/react';

function App() {
  return (
    <HumanBehaviorProvider apiKey="your-api-key-here">
      <YourApp />
    </HumanBehaviorProvider>
  );
}

function MyComponent() {
  const tracker = useHumanBehavior();
  
  const handleButtonClick = async () => {
    await tracker.customEvent('button_clicked', {
      buttonId: 'my-button',
      page: window.location.pathname
    });
  };
  
  return (
    <button onClick={handleButtonClick}>
      Click me
    </button>
  );
}
```

## What Happens Next?

Once you've completed the setup:

1. **Automatic Session Recording**: The SDK automatically captures user interactions using rrweb
2. **Session Persistence**: Sessions persist across page reloads for 30 minutes
3. **Navigation Tracking**: SPA navigation and URL changes are automatically tracked
4. **Console Tracking**: Console logs, warnings, and errors are captured
5. **Data Processing**: Events are processed and sent to your ingestion server
6. **User Identification**: Users are automatically identified using cookies
7. **Analytics**: View session recordings and analytics in your HumanBehavior dashboard

## Key Features

The SDK automatically provides:

- **Session Recording**: Mouse movements, clicks, scrolls, form inputs
- **Navigation Tracking**: SPA route changes, browser navigation
- **Console Tracking**: Log, warn, error messages
- **Custom Events**: Track specific user actions
- **Data Redaction**: Protect sensitive information
- **Anti-blocking**: Handles ad blockers gracefully
- **Session Continuity**: Persists across page reloads

## Troubleshooting

### Common Issues

**Tracker not initializing**: Make sure your API key is valid and you have an internet connection.

**Events not being sent**: Check your browser's network tab for any failed requests to the ingestion server. The SDK includes anti-blocking measures for ad blockers.

**User identification failing**: Ensure you're calling `addUserInfo()` after the tracker has initialized.

**Session not persisting**: The SDK automatically persists sessions for 30 minutes. Check that localStorage is available in your browser.

### Connection Issues

If you're experiencing connection issues:

```javascript
// Test connection
const status = await tracker.testConnection();
console.log('Connection successful:', status.success);

// Get connection status
const connectionInfo = tracker.getConnectionStatus();
console.log('Blocked:', connectionInfo.blocked);
console.log('Recommendations:', connectionInfo.recommendations);
```
